## مصطلحات تقنية إضافية (من الكورس)

| المصطلح | المعنى |
|---------|--------|
| 1A | القمة اللي بتتكوّن بعد اختراق منطقة نقطة الانعكاس بالكامل (نقطة الدخول الثاني) |
| Spring | الكسر الكاذب — بيحصل في الدعوم |
| Upthrust | الاختراق الكاذب — بيحصل في المقاومات |
| نموذج السوبر | نموذج شراء: هبوط لدعم → كسر كاذب → عودة فوق الدعم → إعادة اختبار → أول شمعة شراء |
| الإزاحة / Unit Price | تحريك العلامة العشرية لضبط حجم الدورة السعرية في أدوات جان |
| الفاصل الزمني | نقطة زمنية متوقع عندها أكشن (صعود/هبوط) — تسريع أو عكس للحركة |
| مربعات الحركة | المربعات داخل الشكل الهندسي الزمني = دعوم ومقاومات للتداول |
| فك الارتباط | خروج السعر من مثلث الحركة في النجمة (يحدد استكمال أو انعكاس) |
| Periodic Weight | الوحدة السعرية الزمنية الخاصة بكل عملة/سهم عند جان |
| PS / SC / AR / ST / SOS / LPS | مراحل نموذج التجميع عند وايكوف (راجع `02_technical_analysis/wyckoff.md`) |
| Divergence / Convergence | تباعد/تقارب بين السعر والمؤشر (راجع `02_technical_analysis/indicators.md`) |
