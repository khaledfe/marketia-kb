# Router — منطق استدعاء المعرفة المشروطة

> الـ Router = خطوة أولى خفيفة (call على Claude Vision) بتبص على الصورة وترجّع **tags** للعناصر الظاهرة فقط.
> بعدها الـ workflow يجيب الملفات المشروطة المطابقة ويحقنها **مع النواة** في call التحليل.

---

## مبدأ التشغيل

1. **النواة (`technical-knowledge-core.md`) تتحقن دايماً** — في كل تحليل بدون استثناء.
2. الـ Router يبص على الصورة ويرجّع tags بصرية (مش تحليلية).
3. لكل tag → يتحقن الملف المشروط المطابق له (بالإضافة للنواة).

---

## برومبت الـ Router (الـ call الخفيف)

```
أنت Router بصري. مهمتك تبص على صورة الشارت وترجّع JSON فيه العناصر المرسومة الظاهرة فقط.
لا تحلل، لا تفسّر — بس حدد إيه الأدوات المرسومة على الصورة.

أرجع JSON بالشكل ده بالظبط (بدون أي نص قبله أو بعده):
{
  "star_or_square": true/false,   // نجمة سداسية أو مربع جان (Gann Box / Square of 9)
  "chart_patterns": true/false,   // مثلث / وتد / علم / رأس وكتفين / قمتين / قناة / ماسة
  "indicators": true/false,       // RSI / MACD / متوسطات / Pivot / Ichimoku / Bollinger ظاهرة
  "seasonal_cycles": true/false,  // فواصل زمنية / تواريخ موسمية / زوايا أفقية / فيبو زمني
  "elliott": true/false,          // ترقيم موجي (1-2-3-4-5 / A-B-C)
  "downtrend": true/false         // الاتجاه العام للشارت هابط بوضوح (قمم وقيعان نازلة)
}

لو مش متأكد من عنصر، رجّعه false.
```

---

## جدول الربط (tag → ملف مشروط)

| الـ tag | الشرط البصري | الملف المحقون |
|---|---|---|
| `star_or_square` | نجمة سداسية / مربع جان / أشكال هندسية زمنية | `conditional/cond-gann-shapes-star.md` |
| `chart_patterns` | مثلث / وتد / علم / رأس وكتفين / قمتين / ماسة | `conditional/cond-chart-patterns.md` |
| `indicators` | مؤشرات ظاهرة تحت أو على الشارت | `conditional/cond-indicators.md` |
| `seasonal_cycles` | فواصل/تواريخ موسمية / زوايا أفقية / فيبو زمني | `conditional/cond-cycles-seasonal.md` |
| `elliott` | ترقيم موجي إليوت | `conditional/cond-elliott.md` |
| `downtrend` | الاتجاه العام هابط بوضوح | `conditional/cond-grid-capital.md` |

ملاحظات:
- `cond-review-rules.md` اتنقل **للنواة الثابتة** (قسم 8) — بقى يتحقن دايماً، مش مشروط.
- ممكن أكتر من tag يبقى true → يتحقن أكتر من ملف مع بعض.
- لو كل الـ tags false → النواة بس (وده طبيعي لشارت بسيط في اتجاه صاعد/عرضي).

---

## التكلفة

- call الـ Router: صغير جداً (صورة + JSON قصير) — أجزاء من السنت.
- الفايدة: التحليل ما بياخدش معرفة متخصصة مالهاش لزمة (تركيز أعلى + توكنز أقل في الحالة المعتادة).
