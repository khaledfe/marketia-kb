# تحليل السوق الكلي | القراءة المركبة

**Category:** Technical Analysis / Market Structure
**المصدر:** market-composite-reading (بدون تغيير جوهري)

---

## المؤشرات الأربعة

- **USDT.D (دومينانس الدولار):** يطلع = ضعف العملات / ينزل = قوة العملات.
- **BTC.D (دومينانس البيتكوين):** يطلع = السيولة في BTC مش البديلة.
- **Total 3 (سوق البديلة بدون BTC وETH):** مقياس موسم البديلة.
- **Altseason Index:** مقياس مباشر لقوة البديلة.

## القراءة المركبة — الحالات الأربعة

1. **USDT.D هابط + BTC.D هابط** = موسم البديلة (Altseason).
2. **USDT.D هابط + BTC.D صاعد** = صعود BTC لوحده.
3. **USDT.D صاعد + BTC.D صاعد** = هروب من البديلة للـ BTC.
4. **USDT.D صاعد + BTC.D هابط** = خروج من السوق كله (كاش).

## تأكيد موسم البديلة (3 شروط مجتمعة)

كسر دعم BTC.D + اختراق قمة Total Market Cap + Altseason Index فوق 75%.

## القاعدة المخرجة

التحليل دايماً بينتهي بقرار محفظة صريح: **زوّد BTC / زوّد البديلة / زوّد الكاش** — مع توضيح السبب.
